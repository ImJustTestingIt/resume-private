# Privacy

People Notes is designed to keep collection explicit and local.

## Data stored

When the user chooses to save a note, the extension may store:

- Person or organization name entered by the user.
- Note text entered by the user.
- Current page URL when the user clicks **Use current page** or **Add selected text**.
- Current page title when the user clicks **Use current page**.
- Selected page text when the user clicks **Add selected text**.

## Data not collected

The extension does not:

- Run automated background collection.
- Track browsing history.
- Scrape social-media pages.
- Send saved notes to a remote service.
- Include analytics or telemetry.

## Storage

Saved notes are stored in Chrome's local extension storage for the current browser profile. The user can delete notes in the popup or export them as JSON.
